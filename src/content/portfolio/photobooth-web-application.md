### Beschreibung

Für meine Hochzeit habe ich mit einem Raspberry Pi, einem Tablet und unserer Spiegelreflex-Kamera einen Photobooth gebaut. Der Raspberry dient als Web-Server auf dem die Web-Applikation liegt. Auf dem Tablet wird dann die Seite aufgerufen und dargestellt. Klickt man dort den Auslöseknopf, so wird das Signal an den Raspberry weitergeleitet. Dieser ist via USB an die Kamera angeschlossen und löst diese aus, empfängt das aufgenommene Bild und schickt dieses an das Tablet zurück, wo man es sich ansehen kann.

Alle Bilder werden auf dem Raspberry gespeichert. Über einen Button auf dem Tablet kann man die Galerie öffnen und sich alle bisher gemachten Bilder ansehen. Zudem ist noch ein QR-Code Feature eingebaut mit dem man sich das Bild, sofern man im gleichen Netzwerk/Wlan ist, das Bild herunterladen kann.

[Hier gibt es die Anleitung, um euch einen eigenen Photobooth zu bauen.](https://www.andrerinas.de/tutorials/raspberry-pi-einen-dslr-weddingphotobooth-erstellen.html "Opens internal link in current window")
