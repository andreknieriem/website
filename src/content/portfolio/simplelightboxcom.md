### Beschreibung

Mein Javascript Plugin SimpleLightbox ist mitlerweile recht beliebt und ich dachte mir, dass das Plugin eine eigene Webseite verdient hat. Die Seite ist zwar "nur" eine Github One-Pager Seite, beinhaltet aber eine Dokumentation und eine Demo der Lightbox in einer neuen ansehnlichen Optik. 

Schaut euch die Seite gerne einmal an! Ich bin für Feedback offen :)
