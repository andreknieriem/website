[![](http://andreknieriem.de/wp-content/uploads/2011/10/android-400x267.jpg "android")](http://andreknieriem.de/wp-content/uploads/2011/10/android.jpg)Heute kam ganz plötzlich die Meldung, dass es eine "App" gibt, die Android auf dem HP Touchpad installiert. Das Ganze lässt sich via Preware installieren. Auf dem Homescreen kommt dann einfach ein Icon. Startet man die App, wird WebOS beendet wie es scheint und Android startet. Leider ruckelt alles und es geht so gut wie gar nichts vernünftig. Es kommt einem einfach so vor, als würde Android in einer virtuellen Maschine emuliert werden. Ob dem so ist kann ich nicht genau sagen. Auf jeden Fall sieht man schonmal, was alles möglich ist und sein wird. Ich denke aber, dass das Release von Cyanogenmod 7 diese App um Meilen übertreffen wird.
