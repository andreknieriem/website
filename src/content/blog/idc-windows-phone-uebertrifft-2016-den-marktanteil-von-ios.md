11\. Jun 2012

[![idc windows ueberholt ios 2016](http://andreknieriem.de/wp-content/uploads/2012/06/idc_windows_ueberholt_ios_2016-700x498.jpg "idc windows ueberholt ios 2016")](http://andreknieriem.de/wp-content/uploads/2012/06/idc_windows_ueberholt_ios_2016.jpg)

Laut IDC soll der Marktanteil von Windows Phone bis 2016 um 46,2 pro Jahr steigen. Also soll der Marktanteil von 5,2 Prozent auf 19,2 Prozent gesteigert werden. Begründet wird das Ganze durch das kommende Betriebssystem Windows 8, dass auch Tablet tauglich ist und demnächst auf sehr vielen Geräten installiert sein soll.

Wenn der IDC recht hat, dann wird Windows Phone bald einen größeren Marktanteil haben als Apples iOS. Googles Android Betriebssystem wird aber weiterhin den Markt anführen.
