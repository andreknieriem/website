[![](http://andreknieriem.de/wp-content/uploads/2011/10/OmniTouch-400x266.png "OmniTouch")](http://andreknieriem.de/wp-content/uploads/2011/10/OmniTouch.png)Microsoft Research, Microsofts Forschungszentrum stellt diese Woche eine neue Technik namens OmniTouch vor. Mit dieser Technik ist es möglich beliebige Oberflächen als Touchscreen zu benutzen.

Umgesetzt wurde das ganze mit einem Schulterprojektor, sowie einer Abwandlung von Microsofts controllerlosen Xbox360 Steuerung Kinect.

Der Projektor projeziert das Bild auf eine beliebige Oberfläche und die Kinect Kamera erkennt die Bewegungen der Hand etc. Microsoft-Forscher Hrvoje Benko sagte, dass das Team die großen Oberflächen nutzen will, die die Welt bietet.

Für einen praktischen Einsatz müsste das ganze noch verkleinert werden, doch schon jetzt kann man von Möglichkeiten träumen, die mit so einer Technik möglich wären.
