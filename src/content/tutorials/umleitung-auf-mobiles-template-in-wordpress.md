Wordpress-Blogs für mobile Geräte wie das iPhone anzupassen ist mit Plugins ziemlich einfach.

Ein gutes Beispiel hierfür ist [Wptouch](http://wordpress.org/extend/plugins/wptouch/). Man installiert das Plugin und hat seinen Blog sofort bereit für das iPhone.

Leider kann man diese Template meist sehr schwierig anpassen. Ein paar Farbanpassung sind zwar meistens möglich, sein Logo kann man auch oben reinsetzen, nur leider sehen diese Seiten dann alle gleich aus, eben wie aus einem Baukasten.

Wer es da individueller mag, für den hab ich etwas gefunden.

Das Plugin heißt [xili-theme-select](http://dev.xiligroup.com/?p=123).

**Was macht dieses Plugin?**

[![xilitheme](http://andreknieriem.de/wp-content/uploads/2011/04/xilitheme-400x289.jpg "xilitheme")](http://andreknieriem.de/wp-content/uploads/2011/04/xilitheme.jpg) Das Plugin erkennt den User Agent des iPhones, bzw. des iPod Touch's und leitet einen sofort auf ein einstellbares Template weiter. Die Einstellungen kann man im Wordpress Backend ganz einfach vornehmen. (siehe Bild rechts).

Nachdem ihr also euer Template eingestellt habt, müsst ihr nur noch ein für das iPhone optimiertes Template bauen.

Ein paar Javascript Schnipsel, Webkit Css Optionen und so weiter werde ich in einem späteren Tutorial erläutern.
