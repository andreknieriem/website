  Ich bin ein stolzer Besitzer eines HP Touchpads, dass mitlerweile Android
  4.1.2 besitzt. Mir gefällt das Design von Windows 8 oder Windows RT und da es
  doch (noch) keine Möglichkeit gibt, dies auf dem Tablet zum Laufen zu bringen,
  muss man sich mit Themes behelfen. Ein User namens brobot175 hat im XDA-Forum
  ein cooles Theme, bzw. eine Anleitung veröffentlicht, die es einem ermöglicht
  das Windows Design bestmöglich auf das Touchpad zu bringen. Ich habe mir mal
  die Mühe gemacht und hier ein kleines Tutorial geschrieben:

  Schritt 1 - Download der benötigten Resourcen:

  Für den PC

  MetroUI Designs

  Mehr Icons, zum Teil von mir gestaltet

  Aus dem Playstore

  ADW Launcher

  Desktop VisualizeR

  Schritt 2 - Dateien auf das Tablet ziehen

  Schließt euer Touchpad an den PC/Mac an und kopiert den Ordner MetroUI auf die
  SD-Karte, also das normale Verzeichnis eures Touchpads

  Schritt 3 - ADW Launcher einrichten

  In der Zip Datei oben befindet sich eine Konfigurationsdatei, die mit dem ADW
  Launcher und ADW Launcher Ex umgehen kann. Dann ist schon alles fertig
  eingestellt. Ansonsten öffnet die XML Datei mit einem Texteditor und schaut
  auch die Werte. Man sollte in den Menüs jeden Punkt von dort wiederfinden.

  Schritt 4 -  Desktop VisualizeR einrichten

  Um uns etwas Zeit zu sparen, sollten wir Desktop VisualizeR Voreinstellungen
  geben. Öffnet dazu den Desktop VisualizeR, klickt auf den Menu Button und dann
  auf Settings. Geht in die Default Settings. Dort stellt ihr folgendes ein:

  Touch Effect : None

  Default Vibration : On

  Text Colour : Egal, denn den Text sollte man für jedes Tile löschen

  Background Colour : Transparent

  Widget Usage : Wählt alle ab bis auf das 4x4.

  Schritt 5 - Kacheln/ Tiles erstellen

  Nun kommt der mühselige Akt.

  - Klickt auf eurem Homescreen etwas länger und es sollte sich die ADW Config
  öffnen

  - Wählt Hinzufügen und dann Widgets

  - Im Menu links wählt ihr dann Desktop VisualizeR und dann das 4x4 Widget

  - Danach sollte auf eurem Home-Bildschirm ein Desktop VisualizeR Icon sein.

  - Klickt auf das Icon und wählt oben bei "Select Icon" "Image file" und
  navigiert zu eurem Metro Ordner

  - Wählt dort Categories für die Überschriften (Start, Tools, Apps, etc),
  smalltiles für die quadratischen und bigtiles für die rechteckigen aus

  - Als Beispiel nehmen wir die Categorie "Start". Also geht in den Ordner
  Categories und wählt dort Start aus. Bei Select Action wählt ihr dann "None"
  aus und klickt auf clear Text, da wir keine Labels brauchen und dann unten auf
  OK

  - Nun sollten wir eine Überschrift haben. Haltet sie gedrückt und wählt im
  Menu dann "Resize" aus. Danach solltet ihr die Größe auf 2Hoch und 3Breit
  machen.

  - Um dasselbe für Apps zu tun macht ihr alles wie bei Categories, doch wählt
  dann bei "Select Action" "Launch Application" aus und sucht eure App aus der
  Liste

  - Smalltiles sollten 1Breit und 2Hoch und Bittiles 2Breit und 2Hoch.

  - Das Ganze macht ihr dann für jede App die ihr haben wollt und füllt so euren
  Bildschirm. Falls es kein Icon gibt, für eure App, dann könnt ihr mit den
  PSD-Dateien im Metro Ordner euch eigene erstellen. Ich biete auch an welche zu
  erstellen.

  Schritt 6 - Spaß haben!

  Das wars auch schon!

  PS. Es gibt noch ein paar Widgets die in das Design gut reinpassen, wie eine
  Uhr etc. Falls Interesse besteht, kann ich diese ja nochmal heraussuchen.
