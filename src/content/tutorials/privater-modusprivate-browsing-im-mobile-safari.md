[![Privater Modus im Mobile Safari](http://andreknieriem.de/wp-content/uploads/2011/07/IMG_1097-200x300.png "Privater Modus im Mobile Safari")](http://andreknieriem.de/wp-content/uploads/2011/07/IMG_1097.png)Jeder hat sicherlich schonmal die Funktion des Privaten Modus, bzw. des Private Browsing im Mobile Safari vermisst. Natürlich gibt es Browser von 3. Anbiertern, die die Funktion unterstützen, jedoch taugen die Browser meistens relativ wenig. Chpwn, eine populärer Entwickler für Jailbreak Apps und Verbesserungen bringt genau dieses Feature in den Mobile Safari!

Das Programm heißt Covert, welches den Button bei der Tabauswahl hinzufügt (siehe Screenshot). Einfach auf den Button klicken und das private Surfen genießen.

**Wichtig**

Für dieses Feature benötigt ihr ein Jailbreak. Danach kann das Programm ganz einfach bei Cydia gefunden werden.
